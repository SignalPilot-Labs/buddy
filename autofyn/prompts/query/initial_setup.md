## First-round setup

This is round 1 of this run. Before any real work:

1. Read `CLAUDE.md`, `README.md`, `package.json`, `pyproject.toml` — understand the project.
2. Install dependencies: `npm ci` for Node projects, `pip install` for Python. Match what the project docs say.
3. Verify the build works. If it fails, fix it before any feature work.

Do this once, at the start of round 1 only. Subsequent rounds inherit the installed environment.
